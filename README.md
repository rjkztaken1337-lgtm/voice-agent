# voice-agent

Голосовой ассистент с активацией по имени ("Привет, Рэс") и клонированным голосом (Coqui XTTS v2). Локальные STT (mlx-whisper) и VAD (webrtcvad) для непрерывного прослушивания без кнопки. Мозг агента — Claude Code CLI в headless-режиме, работающий через прокси-сервис (не прямой платный Anthropic API), поскольку из страны пользователя нет возможности оплатить его напрямую.

## Установка

1. Python 3.11 через Homebrew (`brew install python@3.11`), venv в `.venv/`.
2. `pip install -r requirements.txt`.
3. Положить чистую запись голоса друга (10–30 сек, без шума/музыки, WAV) в `voice_sample/friend.wav`.
4. Убедиться, что Claude Code уже авторизован (обычный вход в VS Code расширение/CLI).
5. Создать `.env` (не коммитится, см. `.gitignore`) с ОТДЕЛЬНЫМ ключом прокси для `agent.py`, не тем же, что использует интерактивная сессия Claude Code — если оба процесса делят один ключ, прокси иногда отвечает "Not logged in" при одновременном обращении:
   ```
   ANTHROPIC_API_KEY=cap_...
   ANTHROPIC_BASE_URL=https://claude-code-cli.vibecode-claude.online
   ```
6. При первом запуске macOS спросит разрешение на микрофон — разрешить.

## Запуск

```
source .venv/bin/activate
python main.py
```

Агент постоянно слушает микрофон через VAD. Скажи "Привет, Рэс" (можно сразу с командой, например "Привет, Рэс, который час") — ассистент ответит голосом друга.

## Как устроен "мозг"

`agent.py` вызывает нативный бинарник Claude Code CLI (`claude -p ...`) в JSON-режиме на каждый ход, с `--permission-mode bypassPermissions` (иначе диалоги подтверждения инструментов не с кем показывать — нет интерактивного терминала) и встроенными инструментами `Bash,Read,Write,WebSearch`. Память между перезапусками — через `--session-id`/`--resume` самого CLI (id сессии хранится в `db/cli_session_id.txt`), без отдельной SQLite-истории.

## Безопасность

Раз подтверждения инструментов на уровне ОС отключены (`bypassPermissions`), защита от опасных команд (rm, sudo, force-push и т.п.) реализована через системный промпт: агент обязан сначала словами описать, что собирается сделать, и дождаться голосового "да" в следующей фразе, прежде чем реально вызвать инструмент. Это программная, а не аппаратная защита — полагается на то, что модель следует инструкции.
